## 💻 Local Development

1. **Clone repository**

   ```bash
   git clone https://github.com/KMV531/green-wave-technologies.git

   ```

2. Install dependencies:

   ```bash
   npm install

   ```

3. Run development server:
   ```bash
   npm run dev
   ```
